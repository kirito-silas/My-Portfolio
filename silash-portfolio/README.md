# Silash Trikhatri — Portfolio

Personal portfolio website for Silash Trikhatri, Procurement Analytics & Business Intelligence professional.

## 🚀 Deploy to GitHub Pages

1. Create a new repo named `kirito-silas.github.io` (or any name like `portfolio`)
2. Upload all files in this folder to the root of the repo
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Your site will be live at `https://kirito-silas.github.io`

## ✅ Before you go live — update these:

| File | What to update |
|------|---------------|
| `index.html` | Replace `your@email.com` with your actual email |
| `index.html` | Replace LinkedIn URL with your profile link |
| `assets/resume/` | Add your `resume.pdf` to this folder |
| `script.js` | Add any additional GitHub repo names to `ALLOWED_REPOS` |

## 📁 Structure

```
silash-portfolio/

├── style.css           ← All styles
├── script.js           ← Animations + GitHub API
└── assets/
```
